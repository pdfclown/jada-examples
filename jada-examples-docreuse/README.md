# Javadoc fragments reuse example

This module demonstrates how to use [Jada taglets](https://github.com/pdfclown/jada/blob/main/jada-ext/src/main/java/org/pdfclown/jada/ext/taglet/DocReuseTaglet.java) and [JadaMaven](https://github.com/pdfclown/jada/tree/main/jada-maven-plugin) to reuse Javadoc fragments and keep them synchronized.
